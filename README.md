# Holdly

个人资产持有成本追踪应用。追踪一次性购买和订阅两种资产类型的每日持有成本。

## Tech Stack

React Router v7 (Remix mode) · Vite · TypeScript · shadcn/ui · Tailwind v4 · Drizzle ORM · Supabase · TanStack Query · Zustand

## Quick Start

```bash
pnpm install
cp .env.example .env.local   # fill in Supabase credentials
pnpm dev
```

## Scripts

| Command | Description |
|---|---|
| `pnpm dev` | Start dev server |
| `pnpm build` | Production build |
| `pnpm lint` | Lint check |
| `pnpm test` | Run tests |
